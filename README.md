# note-mark

A MarkDown editor from a Youtube tutorial.

## Youtube Tutorial

- [Tutorial](https://youtu.be/t8ane4BDyC8?si=TIq395ZQf6NW6J4i)

## Project Setup

### Install

```bash
$ yarn
```

### Development

```bash
$ yarn dev
```

### Build

```bash
# For windows
$ yarn build:win

# For macOS
$ yarn build:mac

# For Linux
$ yarn build:linux
```
